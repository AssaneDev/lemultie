Prompt global amélioré – Projet Multi-Voix
Contexte

Créer une plateforme digitale moderne, immersive et performante pour Multi-Voix, une entreprise spécialisée dans :

Copywriting (événements, discours de mariage, associations)
Voix off professionnelle
Percussions et rythmes
Méditations guidées (audio & vidéo)
Contes et storytelling
Formations (oralité, rythmique, expression)
Conférences
Objectifs produit
Centraliser toutes les offres dans une plateforme cohérente
Offrir une expérience immersive (type VSL / storytelling)
Permettre la monétisation (formations, abonnements, contenus premium)
Créer une forte identité visuelle autour de la voix, du rythme et de l’émotion
Contraintes techniques
Architecture découplée (Frontend / Backend)
Backend : Laravel
Frontend : Vue.js + Inertia.js
Application en PWA (Progressive Web App)
API REST ou Inertia-driven
Expérience utilisateur
Design épuré, moderne, immersif
Background uniforme (type landing VSL)
Navigation fluide, orientée conversion
Mise en avant de l’audio (lecteur intégré, expérience sensorielle)
Mobile-first
Modules clés
Homepage immersive (branding + storytelling)
Catalogue de services
Plateforme de formations
Espace abonnés (contenus premium)
Méditations audio/vidéo
Blog / conférences
Système de paiement
Dashboard admin
Identité visuelle
